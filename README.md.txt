1️⃣ Project Title
Netflix Content Analysis Dashboard (Excel)

2️⃣ Project Overview
This project analyzes Netflix movies and TV shows using Excel.
The goal is to transform raw data into meaningful insights through
charts and visualizations.

3️⃣ Dataset Description
Dataset Name: Netflix Titles Dataset

Columns:
- Show_id: Unique ID of content
- Type: Movie / TV Show
- Title: Name of the content
- Country: Country of production
- Release Year: Year released
- Rating: Content rating
- Duration: Movie length or TV seasons
- Listed_in: Genre

4️⃣ Tools Used
- Microsoft Excel
- Pivot Tables
- Bar Chart Visualization

5️⃣ Data Cleaning & Preparation
- Removed blank values in Country
- Standardized country names
- Verified data types
- Prepared pivot table for analysis

6️⃣ Exploratory Data Analysis (EDA)
Key questions answered:
- Which country has the highest Netflix content?
- How is content distributed across countries?

7️⃣ Dashboard / Visualization
- Bar chart showing Netflix content by country
- Countries: United States, India, Spain, Germany
- Metric: Count of Titles

8️⃣ Insights & Findings
- United States has the highest number of Netflix titles
- India and Spain follow with moderate content count
- Germany has comparatively fewer titles

9️⃣ Conclusion
This analysis shows Netflix’s strong focus on the US market.
Excel dashboards help convert raw data into clear business insights.
